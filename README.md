# cpp_rush00
cpp_piscine_rush00

The goal is implement (in team of two students) a simlistic shoot-em-up-style game in terminal until 2 days is end.

You can move and shots until ammo is not end.
Enemy is randomly move and shoots.

**legends:**
```
"@" - player;
"#" - enemy;
"^" - player bullets;
"v" - enemy bullets;
```
**features:**
```
  colission detected 
    by map edge 
    you can ram enemies at the cost of 1
    enemy shoots on you and you may shot him
  Tree levels with boss "$" fight at the end. Boss hase lots of HP.
  HP counter
  Bullets counter
  Score counter
  Game timer
```
